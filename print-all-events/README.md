# K2's CC Repo / Print All Methods
## Usage:
This program works both advanced and normal computers and monitors. <br>
<br>
Place the computer and then run the program, once any event fires it will be printed in the terminal. <br>
<br>
## Installation:
Run this command in the terminal to import the program:
> wget [https://raw.githubusercontent.com/K2568/K2s-CC-Repo/refs/heads/main/print-all-events/startup.lua](https://raw.githubusercontent.com/K2568/K2s-CC-Repo/refs/heads/main/print-all-events/startup.lua)
